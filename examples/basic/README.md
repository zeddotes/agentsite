# Basic example

Minimal Vite + React app using Agentsite.

```bash
# from repo root
npm install
npm run build

# then
cd examples/basic
npm install
npm run dev
```

Open the URL Vite prints. Humans see the interstitial; append `?visible=1` for the full UI. Inspect `#agentsite-json-twin` in the DOM for the machine payload.
